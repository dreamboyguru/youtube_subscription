# YouTube Subscription

This backend project is used to retrieve data from the database by utilizing various APIs 

## Deployment

To deploy this project run

```bash
  https://youtube-subscription.onrender.com
```

## Start command

To start this project run

```bash
  $node src/index.js
```


## Build command

```bash
  $npm install
```
